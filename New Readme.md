Project Overview

Developed an HR Analytics Dashboard in Power BI to analyze employee attrition trends across departments, age groups, salary bands, genders, and job roles. The dashboard provides interactive insights to support workforce planning and retention strategies.

Tools Used:

Power BI
Power Query
DAX
Excel


KPIs:

Total Employees
Attrition Count
Attrition Rate
Average Salary
Average Age
Average Tenure


Dashboard Analysis:

Attrition by Department
Attrition by Gender
Attrition by Age Group
Attrition by Salary Band
Attrition by Job Role


Key Findings:

Attrition Rate = 14.8%
Finance and HR show highest attrition.
Age Group 30-39 shows highest attrition.
High salary employees contribute significantly to attrition.
Reporting Analysts and Decision Analytics Associates show highest attrition.
